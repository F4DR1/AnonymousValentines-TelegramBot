# Telegram-бот с анонимными сообщениями и платежами

Бот позволяет пользователям обмениваться анонимными сообщениями. Реализована интеграция с ЮKassa для приёма платежей.

## Технологии
- Python 3.x
- Flask (для WebHook)
- requests
- ЮKassa API

## Установка и запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Создайте файл `.env` и укажите там токен бота и ключи ЮKassa.
3. Запустите сервер: `python app.py`
4. Настройте WebHook через Telegram API (инструкция в документации).
